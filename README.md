# Onderzoek
Een omgeving waar ik onderzoek doe naar verschillende onderwerpen.<br>
Met vergelijkingstabellen en onderzoeksresultaten. Bootstrap wordt gebruikt om snel dingen weer te kunnen weergeven met een Minimal Reproducible Example (MRE).

## Wat is een Minimal Reproducible Example (MRE)?
Een MRE is een stukje code dat de lezer kan kopiëren en plakken in zijn eigen omgeving om hetzelfde resultaat te krijgen als de schrijver.
Zie uitgebreid uitleg: https://stackoverflow.com/help/minimal-reproducible-example

## Wat is Proof of Concept (PoC)?
Een PoC is een stukje code dat de lezer kan kopiëren en plakken in zijn eigen omgeving om te zien of het werkt.
De termen MRE en PoC worden vaak door elkaar gebruikt. Echter, PoC is vaak uitgebreider dan een MRE.
